< Maya Tools: Playblast tool

A tool to create nice-looking playblasts in Maya

Playblast tool showcase

We wanted a way to output more official-looking playblasts for archive and/or showcase purposes. These are used to, for example, show and get feedback on in-progress and completed animation before implementation, and when creating audio. Knowing the exact frame when things happens makes it a lot easier to both feedback and plan sound effects.

You can include up to 3 different views of the character + props, and these can be positioned wherever wanted. Depending on needs, the output can be 1 single file for all animations, or one file per clip.

As a small quality of life feature, the tool checks if the rig is up to date when opened (and helps update the rig if not!), since my animators tended to do in-progress playblasts relatively frequently, so this became a good way to keep the rig updated in animation scenes.
